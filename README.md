Patient Visit Summary Printing System

Features :-
Generate patient visit summaries
Printable format output
Built with Spring Boot & Hibernate
Architecture :-
Controller → Service → Repository
MVC-based design
Sample Output :-
Patient Name: John Doe
Visit Date: 2026-03-27
Diagnosis: Flu
Purpose :-

This project demonstrates a scalable implementation of patient summary printing aligned with real-world healthcare systems like OpenMRS.
